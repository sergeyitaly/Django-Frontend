1. cd frontend
2. npm install all
3. nmp run build
4. Navigate to the Python downloads page: Python downloads.
5. Install python
5. cd backend
7. python -m pip install Django
8. pip install django djangorestframework django-cors-headers Pillow django-admin-interface django-rest-knox
9. python manage.py runserver
10. npm start
12. Go to http:/127.0.0.1:8000 and see main site page
13. Go to http:/127.0.0.1:8000/admin/ (admin, pass: admin) and see admin page
14. Go to http:/127.0.0.1:8000/products/ and see all Json parsed items
15. Go to http:/127.0.0.1:8000/categories/ and see all Json parsed categories

All data are saved in backend db.sqlite3. See them at admin panel.
